## 📓 **笔记1-500: Python编程**

### 笔记1: 变量和数据类型
**日期**: 2026-03-01
**主题**: Python基础

**学习内容**:
```python
# 变量定义
name = "张三"
age = 25
height = 1.75
is_student = True

# 数据类型
type(name)      # <class 'str'>
type(age)       # <class 'int'>
type(height)    # <class 'float'>
type(is_student) # <class 'bool'>
```

**关键点**:
1. Python是动态类型语言
2. 变量不需要声明类型
3. type()函数查看类型

### 笔记2: 字符串操作
**日期**: 2026-03-02
**主题**: 字符串处理

**学习内容**:
```python
# 字符串格式化
name = "张三"
age = 25

# 方法1: f-string
s1 = f"我是{name}，今年{age}岁"

# 方法2: format
s2 = "我是{}，今年{}岁".format(name, age)

# 方法3: %
s3 = "我是%s，今年%d岁" % (name, age)
```

**关键点**:
1. f-string最简洁（Python 3.6+）
2. format()通用性强
3. %是旧式写法
