## 🎯 核心理念

**参考 autoresearch**:
> 给 AI Agent 一个真实的任务环境，让它自主实验。它修改代码，训练5分钟，检查结果是否改进，保留或丢弃，然后重复。

**应用到学习探索**:
1. **多分支并行** - 开启多个子 agent 同时处理不同任务
2. **自主探索** - 子 agent 独立完成，无需人工干预
3. **结果导向** - 关注产出质量和效率
4. **快速迭代** - 完成后立即整合，继续下一轮

## 💡 经验总结

### **适用场景**
1. ✅ 独立任务（无依赖关系）
2. ✅ 相似结构（可复用模板）
3. ✅ 高价值任务（值得并行）
4. ✅ 资源充足（Token 充足）

### **不适用场景**
1. ❌ 有依赖关系的任务
2. ❌ 需要人工决策的任务
3. ❌ 资源紧张的情况
4. ❌ 复杂交互的任务
