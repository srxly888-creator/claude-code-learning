## 🚀 5分钟快速开始

### **1. 安装 Claude Code CLI**

```bash
# macOS/Linux
curl -fsSL https://claude.ai/install.sh | sh

# 或使用 npm
npm install -g @anthropic/claude-code-cli

# 验证安装
claude-code --version
```

### **2. 配置 API Key**

```bash
# 设置环境变量
export ANTHROPIC_API_KEY="your-api-key-here"

# 或创建配置文件
mkdir -p ~/.config/claude-code
echo "ANTHROPIC_API_KEY=your-api-key-here" > ~/.config/claude-code/.env
```

### **3. 第一个命令**

```bash
# 初始化项目
cd my-project
claude-code init

# 生成代码
claude-code generate --prompt "创建一个简单的 REST API"

# 调试代码
claude-code debug --file app.py --error "TypeError"

# 优化性能
claude-code optimize --focus performance --file app.py
```

## 🎯 常见使用场景

### **场景 1: 新项目启动**

```bash
# 1. 创建项目目录
mkdir my-awesome-project
cd my-awesome-project

# 2. 初始化
claude-code init

# 3. 生成项目结构
claude-code generate --template web-app --name my-awesome-project

# 4. 生成 README
claude-code doc --type readme --project my-awesome-project
```

### **场景 2: Bug 修复**

```bash
# 1. 分析错误
claude-code debug --file app.py --error "TypeError: 'NoneType' object is not subscriptable"

# 2. 生成修复
claude-code fix --issue "NoneType error in line 42"

# 3. 验证修复
python app.py
```

### **场景 3: 性能优化**

```bash
# 1. 分析性能
claude-code analyze --type performance --file app.py

# 2. 优化建议
claude-code optimize --focus memory --file app.py

# 3. 基准测试
claude-code benchmark --file app.py
```

### **场景 4: 代码重构**

```bash
# 1. 分析代码质量
claude-code analyze --type quality --file app.py

# 2. 重构
claude-code refactor --style clean-code --file app.py

# 3. 验证功能
claude-code test --file app.py
```

## 💡 最佳实践

### **1. 提示词技巧**

**❌ 差的提示词**:
```
"写代码"
```

**✅ 好的提示词**:
```
"创建一个 Python Flask REST API，包含以下功能：
1. 用户注册和登录（JWT认证）
2. CRUD 操作（创建、读取、更新、删除）
3. 错误处理和数据验证
4. 单元测试
使用 FastAPI 框架，遵循 PEP 8 代码规范。"
```

### **2. 上下文管理**

```bash
# 提供足够的上下文
claude-code generate \
  --context "这是一个电商平台的后端服务" \
  --context "使用 PostgreSQL 数据库" \
  --context "需要处理高并发" \
  --prompt "创建订单处理系统"
```

### **3. 迭代开发**

```bash
# 第1轮：生成基础功能
claude-code generate --prompt "基础用户管理"

# 第2轮：添加功能
claude-code generate --prompt "添加邮箱验证"

# 第3轮：优化
claude-code optimize --focus performance
```

### **4. 测试驱动**

```bash
# 先生成测试
claude-code test --prompt "测试用户注册功能"

# 再生成代码
claude-code generate --prompt "实现用户注册"
```

## ❓ 常见问题

### **Q1: 如何保护 API Key？**
A: 使用环境变量或配置文件，不要提交到 Git。

### **Q2: 生成的代码质量如何？**
A: 通常很好，但需要人工审查和测试。

### **Q3: 支持哪些语言？**
A: Python, JavaScript, TypeScript, Go, Rust, Java 等。

### **Q4: 费用如何？**
A: 按 token 计费，约 $10-50/月（取决于使用量）。
