# Claude Code 快速開始（繁體中文）

> **版本**: 1.0 | **更新**: 2026-03-22

---

## ⚡ 5分鐘快速開始

### **步驟1: 安裝**（30秒）

```bash
# macOS/Linux
curl -fsSL https://claude.ai/install.sh | sh

# 驗證安裝
claude --version
```

---

### **步驟2: 設定**（30秒）

```bash
# 設定 API Key
export ANTHROPIC_API_KEY="sk-ant-..."

# 驗證設定
claude config check
```

---

### **步驟3: 第一個命令**（1分鐘）

```bash
# 啟動 Claude Code
claude

# 輸入提示詞
建立一個 Python Hello World 程式

# 查看結果
```

**預期輸出**:
```python
def hello_world():
    """列印 Hello World"""
    print("Hello, World!")

if __name__ == "__main__":
    hello_world()
```

---

### **步驟4: Plan Mode**（1分鐘）

```bash
# 1. 啟動 Claude Code
claude

# 2. 按 Shift + Tab
# 切換到 Plan Mode

# 3. 輸入任務
分析當前目錄的程式碼結構

# 4. Claude 會研究程式碼庫並提出計劃

# 5. 審核計劃

# 6. 按 Shift + Tab
# 切換到 Accept Edits 執行
```

---

### **步驟5: 批次處理**（1分鐘）

```bash
# 建立任務檔案
cat > tasks.txt << EOF
建立使用者 API
建立商品 API
建立訂單 API
EOF

# 批次生成
claude batch-generate --file tasks.txt
```

## 🎯 核心概念（30秒）

### **三種模式**

| 模式 | 用途 | 特點 |
|------|------|------|
| **Plan Mode** | 研究程式碼庫 | 唯讀，不修改 |
| **Accept Edits** | 自動修改 | 即時修改 |
| **Review Edits** | 手動審核 | 每個修改需確認 |

### **三個原則**

1. **輸入品質 = 輸出品質**
2. **Planning before Building**
3. **簡單 > 複雜**

## 💡 快速技巧

### **技巧1: 使用範本**（10秒）

```bash
# 建立範本
claude template create api-crud

# 使用範本
claude generate --template api-crud --params "resource=user"
```

---

### **技巧2: 提供範例**（10秒）

```bash
claude generate \
  --prompt "建立使用者 API" \
  --example good_example.py
```

### **技巧3: 明確約束**（10秒）

## 🚨 常見錯誤

### **錯誤1: API Key 無效**

```bash
# 解決方案
export ANTHROPIC_API_KEY="sk-ant-..."
```

---

### **錯誤2: Token 超限**

```bash
# 解決方案
claude generate --max-tokens 1000
```

---

### **錯誤3: 網路逾時**

```bash
# 解決方案
claude config set timeout 60
```
