## 📝 提示词最佳实践

### **最佳实践1: 结构化提示词**

**✅ 好的提示词**:
```markdown
## 背景
电商平台后端服务，使用 FastAPI + PostgreSQL

## 目标
创建用户注册和登录功能

## 约束
- 使用 JWT 认证
- 密码加密存储（bcrypt）
- 邮箱格式验证

## 要求
1. 完整的错误处理
2. 单元测试覆盖
3. API 文档

## 输出格式
- 代码文件（带注释）
- 测试文件
- API 文档（Markdown）
```

**❌ 差的提示词**:
```
写个函数
```

**提升**: **5x**

### **最佳实践5: 迭代优化**

```bash
# 1. 生成基础版本
claude generate --prompt "..."

# 2. 代码审查
claude review --file app.py

# 3. 修复问题
claude fix --file app.py --issues review.md

# 4. 重构
claude refactor --file app.py --style clean-code
```

**效果**: 质量提升 **40%**

---

### **最佳实践6: 测试驱动**

```bash
# 1. 先写测试
claude generate --prompt "为用户API编写测试"

# 2. 再写代码
claude generate --prompt "实现用户API，确保通过测试"

# 3. 运行测试
pytest tests/
```

**效果**: Bug率降低 **80%**
