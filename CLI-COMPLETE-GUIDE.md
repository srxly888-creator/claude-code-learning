## 📋 **什么是Claude Code CLI？**

**CLI** = Command Line Interface（命令行界面）

**简单说**：就是在终端里用文字命令操作Claude Code

**类比**：
- GUI（图形界面）= 用鼠标点击
- CLI（命令行）= 用键盘打字

---

## 🚀 **基础命令**

### 1. 启动Claude Code
```bash
claude
```

**效果**：启动Claude Code交互模式

---

### 2. 单次提问
```bash
claude "你好，请帮我写一个简单的待办清单程序"
```

**效果**：Claude Code会直接给你答案

---

### 3. 指定文件
```bash
claude --file README.md "请解释这个文件的内容"
```

**效果**：Claude Code会读取文件并解释

---

### 4. 指定目录
```bash
claude --dir ./my-project "请分析这个项目"
```

**效果**：Claude Code会分析整个目录

---

### 5. 保存输出
```bash
claude "写一个Hello World程序" > output.txt
```

**效果**：结果保存到output.txt文件

---

## 🛠️ **高级用法**

### 1. 连续对话模式
```bash
claude chat
```

**效果**：进入连续对话模式，可以多轮交互

---

### 2. 指定模型
```bash
claude --model claude-3-opus "请解释量子计算"
```

**效果**：使用Claude 3 Opus模型

---

### 3. 设置温度
```bash
claude --temperature 0.7 "写一首诗"
```

**效果**：控制创造性（0-1，越高越有创意）

---

### 4. 限制Token
```bash
claude --max-tokens 1000 "总结这篇文章"
```

**效果**：限制输出长度

---

### 2. 分步执行
```bash
# 第1步：创建基础结构
claude "创建项目基础结构" > structure.md

# 第2步：添加详细功能
claude --file structure.md "添加详细功能说明" > full-doc.md

# 第3步：生成使用案例
claude --file full-doc.md "生成3个使用案例" > examples.md
```

---

### 3. 迭代优化
```bash
# 初版
claude "创建助理团文档" > v1.md

# 优化
claude --file v1.md "优化文档结构和内容" > v2.md

# 完善
claude --file v2.md "添加更多案例和最佳实践" > v3.md
```
