## 📋 **模板5: API文档**

```markdown
# API文档：[接口名称]

## 接口描述

简要描述接口功能。

## 请求信息

**URL**: \`/api/v1/resource\`
**方法**: \`POST\`
**认证**: Bearer Token

## 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| param1 | string | 是 | 参数说明 |
| param2 | number | 否 | 参数说明 |

## 请求示例

\`\`\`json
{
  "param1": "value1",
  "param2": 123
}
\`\`\`

## 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| code | number | 状态码 |
| message | string | 消息 |
| data | object | 数据 |

## 响应示例

\`\`\`json
{
  "code": 200,
  "message": "success",
  "data": {
    "id": 1,
    "name": "example"
  }
}
\`\`\`

## 错误码

| 错误码 | 说明 |
|--------|------|
| 400 | 参数错误 |
| 401 | 未授权 |
| 500 | 服务器错误 |
```
