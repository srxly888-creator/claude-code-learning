## 📊 **团队使用规范**

### **1. 提示词规范**
```markdown
# 好的提示词示例
✅ 具体明确：
"创建一个用户登录表单，包含邮箱验证和密码强度检查"

✅ 提供上下文：
"这是一个电商项目，使用React和TypeScript"

✅ 遵循规范：
"按照团队代码规范，使用ESLint和Prettier"

# 差的提示词示例
❌ 模糊不清：
"帮我写个表单"

❌ 无上下文：
"登录页面"

❌ 不规范：
```

## 🛠️ **团队工具集成**

### **1. Git集成**
```yaml
# .github/workflows/claude-code.yml
name: Claude Code CI
on: [pull_request]
jobs:
  review:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v2
      - name: Code Review
        run: claude-code review
```

### **2. Slack集成**
```javascript
// Slack机器人
app.command('/claude', async ({ command, ack, respond }) => {
  await ack();
  const result = await claudeCode.generate(command.text);
  await respond(result);
});
```

### **3. Jira集成**
```javascript
// Jira自动化
jira.on('issue.created', async (issue) => {
  const code = await claudeCode.generate(issue.description);
  // 创建代码PR
});
```
