## ❌ **错误1-100: 语法错误**

### 错误1: 缺少冒号
**错误代码**:
```python
if x > 5
    print("大")
```

**错误信息**: `SyntaxError: invalid syntax`

**正确代码**:
```python
if x > 5:
    print("大")
```

**教训**: Python的if、for、while、def都要加冒号。

---

### 错误2: 缩进错误
**错误代码**:
```python
if x > 5:
print("大")
```

**错误信息**: `IndentationError: expected an indented block`

**正确代码**:
```python
if x > 5:
    print("大")  # 4个空格缩进
```

**教训**: Python依赖缩进，必须保持一致。

---

### 错误3: 变量未定义
**错误代码**:
```python
print(x)
```

**错误信息**: `NameError: name 'x' is not defined`

**正确代码**:
```python
x = 10
print(x)
```

**教训**: 使用变量前必须先定义。
